# BST Address Book System
A C++ application developed for CIS 275 that implements a Contact Management System using a Binary Search Tree (BST).

## Features
- **Efficient Operations:** Insertion, Deletion, and Search all perform at O(log n) average time complexity.
- **Alphabetical Ordering:** Utilizes In-order traversal to display contacts sorted by name.
- **Contact Details:** Stores name, phone number, email, and home address.

## Technologies Used
- **Language:** C++
- **Data Structures:** Binary Search Tree (BST)
- **Environment:** Visual Studio / GDB

## How to Run
1. Compile the `main.cpp` file using a C++ compiler (like G++ or Visual Studio).
2. Follow the on-screen menu prompts to manage your contacts.
